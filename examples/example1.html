Quick Notes

  • Haml is usually indented with two spaces, although more than two is allowed. You have to be consistent, though.
  • The first character of any line is called the "control character" - it says "make a tag" or "run Common Lisp code" or all sorts of things.
You can get more information by reading the Official CL-HAML Reference.

This example doesn't allow Common Lisp to be executed, but real CL-HAML does.

Common Lisp code is included by using = at the beginnig of a line.
